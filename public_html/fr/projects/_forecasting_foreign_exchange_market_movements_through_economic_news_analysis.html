Anticipation des Mouvements du Marché des Changes grâce à l’Analyse des Nouvelles Économiques

2020 - 2021

  • Ce projet permet l’analyse sentimentale des nouvelles économiques à l’aide d’un modèle de langage (LLM).
  • Un agent récupère et transforme les nouvelles économiques, qui sont ensuite stockées dans une base de données. Une interface web permet l’annotation manuelle de ces données pour enrichir le jeu d’entraînement. Un LLM préentraîné est alors fine-tuné.
